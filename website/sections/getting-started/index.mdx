<p>
  To get up and running, install chakra-ui-steps using your favorite
  package manager:
</p>

Yarn:

```bash
yarn add chakra-ui-steps
```

NPM:

```bash
npm i chakra-ui-steps
```

## Features

- Multiple orientations
- Easily render step content
- Custom icons
- Size variants
